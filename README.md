# IT Support Portfolio

This repository documents my hands-on learning path toward remote IT support, technical support, Microsoft 365 support, cloud support, and automation-related roles.

## Current Focus

* Windows 10/11 troubleshooting
* Basic networking
* Help desk ticket documentation
* Microsoft 365 support basics
* PowerShell scripting
* Cloud fundamentals
* Remote support workflows

## Portfolio Sections

### Help Desk Tickets

Practice tickets showing how I troubleshoot common IT issues, document steps, identify root causes, and write clear resolutions.

### Windows Troubleshooting

Notes and labs covering common Windows support problems such as updates, printers, drivers, slow performance, user accounts, malware checks, and system tools.

### Networking Basics

Labs and notes covering IP addresses, DNS, DHCP, ping, tracert, ipconfig, Wi-Fi, Ethernet, routers, and basic connectivity troubleshooting.

### Microsoft 365 Support

Documentation for common support tasks involving Outlook, Teams, OneDrive, user accounts, MFA, password resets, and basic admin workflows.

### PowerShell Scripts

Small scripts used to practice automation, system checks, file management, and support tasks.

### Cloud Labs

Beginner cloud projects using AWS, Azure, or related tools, with a focus on documentation, security basics, and cost control.

## Career Goal

My goal is to build the skills needed for remote IT support, SaaS technical support, Microsoft 365 support, cloud support, and eventually systems automation or DevOps-adjacent work.
